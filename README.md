### Zebra-Printer_How to Show Comany Logos During Boot Sequence

---

###  白黒液晶＋十字ボタンモデルでブート中に企業ロゴを表示する方法

1. 企業ロゴを下記フォーマットの画像に保存する。  
    -  Bitmap Mono フォーマットで保存
    -  240 x 128 pixel 
    - ファイル名は"STARTING_UP.BMP"
1. プリンタのEドライブにSTARTING_UP.BMPをコピーする。
1. プリンタを再起動する。

</br>
